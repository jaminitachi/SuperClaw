아침 브리핑을 실행해. 모든 단계를 빠짐없이 수행하고, 겉핥기 하지 말고 각 채팅방/메일을 하나씩 열어서 내용을 직접 읽어.

=== 1단계: 카카오톡 메시지 읽기 ===

1-1. KakaoTalk 앱 activate (AppleScript)
1-2. 메뉴바 '창' > '채팅' 클릭으로 채팅 목록 열기
1-3. sc_screenshot으로 채팅 목록 캡처
1-4. 이미지에서 채팅방 식별:
   - 개인 채팅, 그룹 채팅만 대상
   - 오픈카카오톡 채팅방은 절대 제외
   - 읽지 않은 메시지 표시(빨간 숫자 뱃지)가 있는 채팅방 우선
1-5. 각 채팅방을 하나씩 처리:
   a) 해당 채팅방을 sc_click으로 클릭하여 열기
   b) 채팅방이 열리면 sc_screenshot으로 캡처
   c) 메시지가 많으면 sc_hotkey로 Page Up 키를 2-3회 눌러서 위로 스크롤
   d) 스크롤 후 다시 sc_screenshot으로 캡처
   e) 캡처한 이미지들에서 메시지 내용을 읽고 기록
   f) 다시 채팅 목록으로 돌아가기 (Cmd+[ 또는 뒤로가기)
   g) 다음 채팅방으로 이동
1-6. 최대 10개 채팅방까지 처리 (그 이상은 "그 외 N개 채팅방 미확인" 표시)

=== 2단계: 이메일 읽기 ===

2-1. Mail 앱 activate (AppleScript)
2-2. sc_screenshot으로 받은편지함 캡처
2-3. 이미지에서 새벽/아침 동안 온 이메일 식별
2-4. 중요해 보이는 이메일(읽지 않은 것) 각각:
   a) sc_click으로 해당 이메일 클릭하여 열기
   b) sc_screenshot으로 이메일 본문 캡처
   c) 이메일이 길면 sc_hotkey로 스크롤 다운 후 추가 캡처
   d) 발신자, 제목, 핵심 내용 기록
   e) 뒤로 돌아가서 다음 이메일로
2-5. 최대 10개 이메일까지 처리

=== 3단계: 요약 및 전송 ===

3-1. 한국어로 요약 작성:
   - 카카오톡: 각 채팅방별로 누구와의 대화인지, 핵심 내용 2-3줄
   - 이메일: 각 메일별로 발신자, 제목, 핵심 내용 1-2줄
   - 긴급하거나 답장이 필요한 항목은 ⚠️ 표시
3-2. sc_send_message 도구로 텔레그램에 전송

중요 규칙:
- screencapture -x 대신 반드시 sc_screenshot / sc_click / sc_hotkey 도구를 사용할 것
- 채팅 목록 한 장만 찍고 끝내지 말 것. 반드시 각 채팅방에 들어가서 내용을 읽을 것
- 메일도 받은편지함 목록만 찍지 말 것. 반드시 개별 메일을 열어서 본문을 읽을 것
